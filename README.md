# Bangalore Pincode Explorer

A full-stack web application to search Bangalore pincodes by area name and area names by pincode.

## Features

- Search by pincode
- Search by area name
- REST API backend
- MongoDB database
- Responsive React frontend
- Seed script for sample Bangalore pincode data

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
